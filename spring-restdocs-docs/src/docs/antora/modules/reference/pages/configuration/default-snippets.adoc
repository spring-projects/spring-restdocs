[[configuration.default-snippets]]
= Default Snippets

Six snippets are produced by default:

* `curl-request`
* `http-request`
* `http-response`
* `httpie-request`
* `request-body`
* `response-body`

You can change the default snippet configuration during setup by using the `RestDocumentationConfigurer` API.
